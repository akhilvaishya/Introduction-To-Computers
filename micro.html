What is a Computer?

A computer is an electronic machine that accepts data as input, processes it according to instructions, stores it, and produces meaningful information as output.

History of Computers

The history of computers started with simple calculating devices like the Abacus. Charles Babbage designed the Analytical Engine and is known as the Father of Computer.

Generations of Computers

First Generation

Used vacuum tubes.

Second Generation

Used transistors.

Third Generation

Used Integrated Circuits.

Fourth Generation

Used Microprocessors.

Fifth Generation

Based on Artificial Intelligence.

Basic Components of Computer

  • Input Unit: Keyboard, Mouse
  • CPU: Brain of computer
  • Memory Unit: Stores data
  • Output Unit: Monitor, Printer

Types of Memory

  • CD/DVD
